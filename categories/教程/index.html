使用 Hugo + GitHub Pages 搭建个人博客

Hugo 是一个用 Go 编写的静态网站生成器，以构建速度快著称。配合 GitHub Pages 和 GitHub Actions，可以轻松实现免费的个人博客部署。

为什么选择 Hugo？

Hugo 有几个显著优点：

  1. 极快的构建速度 — 数百篇文章在毫秒级完成构建
  2. 单一二进制文件 — 无需安装依赖，一个文件即可运行
  3. 强大的模板系统 — 基于 Go template，灵活且高性能
  4. 丰富的主题生态 …
